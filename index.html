What's inside?

Silex Bootstrap is nothing more than Silex with some stuff already set up for you:
  • Twitter bootstrap
  • HTML5 Boilerplate
  • Template rendering with Twig
  • Database interface with Doctrine
  • Directory structure for your project
Here is the example controller included in the sources. 
$app->get('/', function() use ($app){
    // Doctrine for database operations
    $app['db.dumb']->insert(array('a' => rand(), 'b' => rand()));

    $data = $app['db.dumb']->findAll();

    // Twig for template rendering
    return $app['twig']->render('index.html.twig', array('data' => $data));
});
And the template. 
{% extends 'layout.html.twig' %}

{% block container_content %}
    <table class="table table-bordered">
        {% for item in data %}
            <tr>
               <td>{{ item.a }}</td>
               <td>{{ item.b }}</td>
            </tr>
        {% endfor %}
    </table>
{% endblock %}

Usage

To start a new project with Silex Bootstrap, just download it, install composer and the dependencies. 
$ cd silex-bootstrap

# Install composer
$ curl -s http://getcomposer.org/installer | php

# Install the dependencies
$ composer.phar install
Edit the database parameters in /src/bootstrap.php and you are done!
